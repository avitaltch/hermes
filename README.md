# Hermes Implementation Helper

A helper project for implementing and working with the [Hermes Agent](https://github.com/nousresearch/hermes-agent) framework by NousResearch.

## Overview

This repository contains utilities, configurations, and examples to streamline development with Hermes Agent. It is built on top of the [prompt-architect](https://github.com/ckelsoe/prompt-architect) skill and leverages free NVIDIA endpoints for experimentation.

## Quick Links

- **Free NVIDIA Endpoints:** https://build.nvidia.com/explore/discover
- **Hermes Agent (GitHub):** https://github.com/nousresearch/hermes-agent
- **Base Skill:** https://github.com/ckelsoe/prompt-architect

## Getting Started

1. Clone this repository.
2. Review the Hermes Agent documentation at the link above.
3. Explore the prompt-architect base skill for foundational concepts.
4. Use the free NVIDIA endpoints to test and iterate on your implementation.
